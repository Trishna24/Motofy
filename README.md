## ✨ Key Features

### User Features
- User Registration & Login
- Browse Available Cars
- View Car Details
- Book Cars Online
- Secure Payment
- Booking History
- AI Chat Assistant

### Admin Features
- Admin Dashboard
- Manage Cars
- Manage Users
- Manage Bookings
- View Analytics
